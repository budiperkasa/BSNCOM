How to update:

1. Copy these files from current package to server:
'index.php' file
'modules' folder
'themes' folder
'updates' folder
'system' folder except 'system/config/database.php' file

2. 'system/config/config.php' file has changed in this version, so you must replace it with new file and then re-enter your domain settings

3. enter such url in browser http://example.com/update/update_v244_to_v251
